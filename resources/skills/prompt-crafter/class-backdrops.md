# Class: backdrops (`backdrop_sparse` / `backdrop_full`)

For `content_class` `backdrop_sparse` / `backdrop_full`, or `type: background`.

## `backdrop_sparse`

- Environmental background with **minimal focal content** — generous empty space for gameplay props / tiles.
- Atmosphere and horizon OK; avoid cluttering the play field.

## `backdrop_full`

- Rich environmental scene; parallax / vista detail as brief describes.
- Full atmosphere — not a sparse placeholder.

## Shared rules

- **No white studio** backdrop; natural environmental lighting.
- No isolated character sprites, no UI overlays, no flat #FFFFFF fill.

## Negatives

No character cutouts on white, no HUD, no tileable-only framing unless brief is texture (use tile class instead).
