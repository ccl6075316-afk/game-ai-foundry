# Class: tiles (`floor_tile` / `wall_tile`)

For `content_class` `floor_tile` / `wall_tile`, or `usage: tile_texture`.

## Technical

- **Seamless tileable texture** filling the frame; uniform lighting; clean tile edges.
- Orthographic / top-down or wall-ortho as appropriate — no perspective hero framing.
- Material and pattern readable at game tile scale.
- **Not** a white studio backdrop — no isolated prop on white.

## Subject

Describe surface material, wear, pattern repeat — not a scene with characters.

## Negatives

No characters, no props, no UI, no white studio character rules, no cast shadows breaking tile repeat.
