# Class: UI (`ui_element` / `icon_kit`)

For `usage: ui_element`, `type: icon_kit`, and HUD/icon craft.

## Technical

- **Orthographic** readable UI — flat color blocks, clear iconography.
- Pure flat white (#FFFFFF) studio for isolated icons / kit items unless brief says in-scene HUD mock (rare).
- Consistent scale; centered single item per generate (`icon_kit` expands one image per `items[]` entry).

## Subject

- HUD chips, buttons, bars: flat design, high contrast at `display_size`.
- Kit items: one object per image — no grids or multi-icon sheets.

## Negatives

No photo-real materials, no perspective UI mockups, no 3D bevel chrome unless art_direction demands stylized flat UI.
No characters, no scenery, no spritesheets.
